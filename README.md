# ABC Clothing Website

This is a premium, responsive website for ABC Clothing, a garment manufacturer based in Cape Town.

## How to View
Simply open `index.html` in your web browser.

## Structure
- **index.html**: Home page with Hero, Services, and Intro.
- **products.html**: Product showcase with grid layout.
- **contact.html**: Contact information and locations.
- **style.css**: Main stylesheet containing the design system (variables, typography, colors).
- **script.js**: JavaScript for mobile navigation.
- **assets/images/**: Generated premium placeholder images.

## Features
- Premium minimal design with a denim-inspired color palette.
- Responsive layout for mobile and desktop.
- SEO-ready meta tags.
- Custom typography (Google Fonts: Outfit & Inter).
